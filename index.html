SwatLauncher

Remote Desktop Connection Manager

Version: 1.0.0

SwatLauncher is a Windows application designed to simplify and enhance the management of Remote Desktop Protocol (RDP) connections. This tool offers a user-friendly interface for IT professionals and system administrators who frequently access multiple remote Windows servers.

Key Features

  • Secure Credential Management: Leverages Windows Credential Manager for secure RDP credentials storage.
  • Active Directory Integration: Automatically populates Windows servers list with FQDNs and descriptions.
  • Efficient Server Management: Easy addition, updating, and deletion of server entries.
  • Quick Search Functionality: Rapid server location through hostname and description filtering.
  • One-Click RDP Connections: Instant RDP session launch with a double-click.
  • System Tray Integration: Minimizes to system tray for clutter-free desktop access.
  • Dark Theme Interface: Reduces eye strain with a modern, sleek UI.
  • Window Position Memory: Remembers last position and size for consistent user experience.

Useful Tips

  • Credential Deletion: Hold shift on the Credential Input window to delete credentials from Windows Credential Manager.
  • CSV file Deletion: Hold shift on the Manage Hosts window to delete the CSV file entirely.
  • Close the Application: Close/minimize the application and right-click the system tray icon to exit.

Technical Highlights

  • Developed using C# and WPF on the .NET Framework.
  • Utilizes Windows API calls for advanced functionality.
  • Stores hostname information in a local 'servers.csv' file for quick access and portability.
